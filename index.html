📄 如何取得在學證明

方法一：使用 NTNU APP 下載（推薦）

  1. 下載並登入「NTNU APP」（國立臺灣師範大學APP）
  2. 進入「我的校園」→「學生專區」
  3. 選擇「教務資訊」→「在學證明」
  4. 選擇當學期並點擊「下載PDF」

💡 APP下載的在學證明已內建教務處電子章戳，無需另外用印！

下載 NTNU APP → 📄 APP操作說明

方法二：i-Portal 網頁下載

請直接點擊連結：https://iportal.ntnu.edu.tw/ntnu/

  1. 登入校務行政資訊入口網
  2. 進入「應用系統」→「日間學制教務資訊系統」
  3. 於左側選單點選「在學證明」並下載

方法三：至教務處申請

  1. 確認已繳完學費
  2. 帶著繳費收據到教務處註冊組申請（費用：NT$10/份）

⚠️ 重要提醒：剛繳完學費的同學，系統可能需要2-3個工作天更新狀態，才能下載在學證明。

🏠 如何取得住宿證明

住學校宿舍：

  1. 確認已繳交住宿費用
  2. 登入校務行政資訊入口網
  3. 進入「學生宿舍相關系統」下載住宿證明

住校外：

請提供與房東簽訂的租賃契約（租屋契約）

步驟二：註冊移民署帳號

觀看影片學習如何註冊移民署線上申辦系統帳號

註冊步驟說明

  1. 前往移民署線上申辦網站：https://reurl.cc/GEbr2p
  2. 點選「註冊新帳號」
  3. 填寫個人基本資料（護照資訊、聯絡方式等）
  4. 設定帳號密碼
  5. 驗證電子郵件
  6. 完成註冊

步驟三：填寫線上申請表

跟著影片一步步完成居留證申請表填寫

申請步驟說明

  1. 登入移民署線上申辦系統
  2. 選擇「外僑居留證申請」
  3. 填寫個人基本資料
  4. 上傳必要文件：
    • 在學證明
    • 住宿證明
    • 2吋照片電子檔
  5. 確認資料無誤
  6. 送出申請

⏰ 居留證效期說明

  • 效期選擇：可選擇1年（NT$1,000）、2年（NT$2,000）或3年（NT$3,000）效期
  • 延期申請：居留證到期前3個月內應申請延展

步驟四：繳費與領證

了解如何完成繳費並領取居留證

繳費步驟說明

  1. 線上申請完成後，系統會產生繳費單
  2. 列印繳費單或記下繳費代碼
  3. 前往便利商店、郵局或銀行繳費（NT$1,000）
  4. 保留繳費收據
  5. 等待審核通知（約7-14個工作天）
  6. 收到通知後，攜帶護照和收據前往移民署領證
